`Tlumx\tlumx-eventmanager` component is PSR-14 event manager implementation.

## Requirements

* PHP >= 7.0

## Installation

Use [Composer](https://getcomposer.org/) to install tlumx-eventmanager.

```bash
$ composer require tlumx/tlumx-eventmanager:"1.0.*"
```
